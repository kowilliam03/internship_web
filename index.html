學習

由於研究主題涵蓋多個領域，因此每個牽涉到的領域都需要有基本的認識，以下會簡單羅列出相關的領域，以其學到的知識，詳細內容會放在書面報告中。

Docker

  • Docker是實作輕量級的作業系統虛擬化的解決方案。
  • Docker的基礎是Linux容器(Linux Containers, LXC)等技術。
  • 學習如何將撰寫好的服務包裝成Docker，並部署至雲端(AWS、Azure)

    • Microservices(微服務)

  • 了解微服務的架構，以及微服務對資訊界產生的影響。
  • 使用基於Python開發的輕量級Web應用框架:Flask，搭建微服務的應用。

    • Data Analysis(資料分析)

  • 了解資料分析的流程，包含:EDA、資料視覺化。
  • 主要使用:Pandas及Numpy進行分析資料，matplotlib及seaborn進行資料視覺化

    • Machine Learning(機器學習)

  • 將指導老師的機器學習講義英翻中，翻譯的過程中，也學到不少東西，像是:機器學習的流程、 EDA的重要性等等。

    • Computer Vision(電腦視覺)

  • 了解Image Processing(影像處理)的基本概念，像是:Filter、Threshold、Edge detection、 Object detection等基本原理及相關的演算法。
  • 使用Scikit-Image套件進行影像處理。

    • 雲端服務

  • 了解雲端服務帶來的改變。
  • 將包裝好的Docker Image部署至AWS及Azure等雲端平台。
